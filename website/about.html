Rotten Reviewers

Visualizing Rotten Tomatoes Reviews.

The data collection, database construction and visualizations for this project were constructed by Jacob Simmering as part of a course project in database management and web services.

The data was obtained by web scraping with Beautiful Soup and Python as well as using the Rotten Tomatoes API. The data was stored in a MySQL database and the dynamic visualizations were produced using the Shiny package and the R Project for Statistical Computing.

More details about how the data was collected, stored or analyzed can be found under the "Methods" tab. All of the code for the project, including this website, can be found on GitHub.
